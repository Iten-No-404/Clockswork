Clockwork

Nowadays, many indie developers create applications and games. These applications face several difficulties to reach the publishing stage and expensive fees are paid. The solution to this problem lies in creating a website that permits all these developers to publish them for free. If the application earns higher than a certain threshold, the developer earns almost all of the value while a very small percentage is taken to keep the website running. The website, called Clockwork, connects between the developers and the target audiences of users for their applications. Clockwork provides an interactive platform for users from all over the world through groups, posts, reviews and comments. Regular users can purchase and download these applications and can try to publish their own creations after the administrators approval.
